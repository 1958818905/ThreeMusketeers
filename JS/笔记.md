3. 函数及面向对象

   - 函数

     - 定义一

       - 绝对值函数

         ```javascript
             function abs(x){
                 if (x>=0){
                     return x
                 }else {
                     return -x
                 }
             }
         
         调用
         abx（10）
         ```

       - return不建议换行，return表示函数结束；如果没有执行return，函数也会执行，结果就是NaN或undefined

     - 定义二

       - 匿名函数

       ```javascript
         var abs1 = function (){
             if (x>=0){
                 return x
             }else {
                 return -x
             }
         }
         
         abs1（10）
       ```
       
     - 参数问题

       - javascript可以传任意个参数，也可以不传

         - `arguments`是一个Js免费赠送的关键字

         - 代表传递进来的所有参数，是一个数组

           ```javascript
           function abs(x){
               for (let i = 0; i < arguments.length; i++) {
                   console.log(arguments[i]);
               }
               if (arguments.length>1){
                   arguments[1] ;
               }
               if (x>=0){
                   return x
               }else {
                   return -x
               }
           }
           ```

         - 问题：arguments包含所有的参数，我们有时候想使用用多余的参数来进行附加操作。需要排除已有参数

           - rest：获取除了当前变量外的参数,只能写在最后面，加...标识

             ```javascript
             function aaa(a,b,...rest){
                 console.log("a=>" + a);
                 console.log("b=>" + b);
             }
             aaa(1,1,2,2)
             ```

       - 不存在参数，规避

         ```javascript
         function abs(x){
             if (typeof x!=="number"){
                 throw "not a number";
             }
             if (x>=0){
                 return x
             }else {
                 return -x
             }
         }
         ```

   - 变量作用域

       ```javascript

           var x = 1;
           x = x + 1;
       }
       ```
